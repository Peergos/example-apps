PaintZ Help

 
Document actions
Clear your current drawing and start over.
Shortcut key: Ctrl+N (only in a standalone window)
Open a new image for editing.
Shortcut key: Ctrl+O
Save your current drawing to your computer.
Shortcut key: Ctrl+S
Print your drawing. You may need to adjust margins or scaling in your system's print preview.
Shortcut key: Ctrl+P
Undo your last change.
Shortcut key: Ctrl+Z
Redo the last change you undid.
Shortcut key: Ctrl+Y
Increase or decrease the canvas size and crop or scale your drawing.
Shortcut key: Ctrl+E
Paste a copied image. Alt-click to “paste from” a file.
Shortcut key: Ctrl+V
 
Tools
Draw single pixels in the current line color. Right-click to use the current fill color.
Shortcut key: P
Draw any shape you want in the current line color. Right-click to use the current fill color.
Shortcut key: B
Spray paint in the current line color. Right-click to use the current fill color.
Draw a straight line in the current line color. Right-click to use the current fill color.
Shortcut key: L
Draw a line in the current line color, then turn it into a curve. Right-click to use the current fill color.
Shortcut key: C
Draw a rectangle. Right-click to swap the line and fill colors.
Shortcut key: R
Draw an oval. Right-click to swap the line and fill colors.
Shortcut key: O
Draw a line, then add additional lines to make a polygon. Double-click or select the starting point to close it. Right-click to swap the line and fill colors.
Draw a rounded rectangle. Right-click to swap the line and fill colors.
Grab and pan around your drawing by clicking and dragging.
Shortcut key: H
Select a rectangular section of your drawing to move (click and drag), duplicate (Ctrl+D), or delete (Delete or Backspace).
Shortcut key: S
Draw a shape to select that section of your drawing to move (click and drag), duplicate (Ctrl+D), or delete (Delete or Backspace).
Shortcut key: F
Cover any part of your drawing with the current fill color. Right-click to use the current line color.
Shortcut key: E
Fill the section with the current line color. Right-click to use the current fill color.
Shortcut key: K
Select a color in your drawing to use it as the line color. Right-click to select the fill color instead.
Shortcut key: I
Add text to your drawing in the current line color.
Shortcut key: T
